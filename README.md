件名：知育向けナンバーリンク（puzzlegame）の継続開発

以前、以下の仕様でHTML5/Canvas製のパズルを制作しました。この設計思想と最新コードをベースに開発を再開したいです。

【1. コア仕様：死守すべきルール】

操作性: 数字の両端どちらからでも線を引け、先端同士が触れると「合流（マージ）」する。数字以外のマスのタップで「1マス消去」が可能。

厳格ルール: 数字は必ず「端点」とする。数字を通り抜けることや、1つの数字から2方向に分岐させることは物理的に禁止。

アルゴリズム（中級）: 「インデックス・シャッフル方式」。1本のハミルトン路を生成後、ルート上は遠いが物理座標が近い箇所をペアにする。これにより、最短距離で結ぶと他を塞ぐ「迂回必須」の難易度を保証。

【2. 実装済みのUI/機能】

難易度（初級・中級）・サイズ・ペア数のバリデーション付き選択。

ギブアップ機能：タイマー領域を「Give Up!」に書き換え、最前面に白点線の強調ガイドを表示する。

クリア演出：全マスを埋めると「Perfect!」ポップアップを表示。
